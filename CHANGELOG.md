# Changelog
All notable changes to this project will be documented in this file.
The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [0.4.1] - 2026-06-30

### ✨ Added
- Full semantic highlighting system for improved cross-language accuracy
- Enhanced support for:
  - JavaScript / TypeScript
  - React (JSX / TSX)
  - CSS / SCSS
  - JSON
  - Python
  - Rust
- New color rules for:
  - default library symbols
  - enums and enum members
  - decorators and namespaces
  - JSX components and props
- Improved CSS syntax coverage:
  - selectors
  - properties
  - pseudo-classes and pseudo-elements
  - units and `!important`
- Improved JSON readability with clearer key distinction
- Added regex and escape character highlighting
- Markdown improvements (headings, links, emphasis)

---

### 🎨 Changed
- Refined overall color system for better consistency across languages
- Improved contrast for light background (`#F7F8FA`)
- Updated:
  - button styles
  - debug console colors
  - chart colors
  - symbol icons
- Improved editor selection, hover, and focus states
- Refined Git diff colors and indicators
- Standardized color palette across UI components

---

### 🛠 Fixed
- Fixed inconsistent enum member coloring
- Fixed visibility issues with certain variables in highlighted states
- Fixed widget shadow artifacts
- Fixed indentation guide color inconsistencies
- Fixed UI border inconsistencies across components

---
